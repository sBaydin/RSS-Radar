GitHub bağlantısı kurulduktan sonra "Şimdi Tara" butonu ve "GitHub'a Kaydet" butonu çalışır hale gelir.
Token nasıl alınır?
1. github.com/settings/personal-access-tokens/new → "Fine-grained token" oluştur.
2. Resource owner: kendi hesabın · Repository access: sadece bu repo.
3. Permissions → Repository permissions: "Contents: Read & write" ve "Actions: Read & write" izinlerini ver.
4. Token'ı kopyala, buraya yapıştır. Sadece tarayıcı localStorage'da saklanır, başka hiçbir yere gönderilmez (GitHub API'ye doğrudan POST).
Değişiklikleri 2 şekilde uygulayabilirsin:
1. YAML indir: Yeni config.yaml'ı bilgisayarına indir, repo'ya kendin commit'le.
2. GitHub'a doğrudan kaydet: 🐙 GitHub sekmesindeki bilgiler kullanılır.
Saat değiştirdiysen workflow dosyası da otomatik güncellenir.
